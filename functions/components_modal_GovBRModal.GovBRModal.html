    Function GovBRModal

    • GovBRModal(__namedParameters: GovBRModalProps): Element

      Modal genérico com estilos e comportamento do padrão GovBR.

      Prefer usar as variantes GovBRInputModal, GovBRConfirmModal ou GovBROptionModal quando formatado para casos específicos.

      Parameters

      • __namedParameters: GovBRModalProps

      Returns Element

      Remarks

      O componente principal (GovBRModal) provê a base visual. As variações (GovBRInputModal, GovBRConfirmModal, GovBROptionModal) adicionam footers e botões prontos para padrões comuns (confirmação, entrada de dados, opções). Os componentes são acessíveis via atributos ARIA já configurados.

      Example

      <GovBRModal open={true} onClose={() => {}} title="Exemplo">
        <div>Conteúdo</div>
      </GovBRModal>

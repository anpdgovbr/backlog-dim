      Converte um objeto ProcessoOutput (do pacote shared-types) para o formato usado pelo formulário de edição/criação (ProcessoFormData).

      Observações:

      • Faz a conversão de campos opcionais para valores primitivos esperados pelo formulário (ex.: ids como number ou null).
      • Converte strings de datas para objetos Date quando aplicável.
      • Mantém statusInterno como opcional, podendo ser StatusInterno | null.
